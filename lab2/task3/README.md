# Лабораторная работа №2 - задача 3: Число инверсий

## Описание
Количество инверсий в последовательности в некотором роде определяет,насколько близка данная последовательность к отсортированной. Задача — подсчитать число инверсий в нем. 

## Структура проекта
- `src/` — исходный код программы.
- `tests/` — автоматические тесты для проверки работы кода.

## Запуск проекта
1. Функция merge(): объединяет две отсортированные половины массива и подсчитывает инверсии между ними.   
2. Функция merge_sort() возвращает общее количество инверсий, найденных в текущем подмассиве
3. Функция count_conversions(): создает временный массив той же длины, что и arr. Она вызывает merge_sort(), чтобы начать рекурсивный подсчет инверсий. 
4. Входные данные взяты из файла input.txt, использую алгоритм сортировки слиянием. Результаты записываются в файл output.txt
